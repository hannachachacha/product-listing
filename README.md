# Product Listing Page

This is a simple **Product Listing Page** built with **React**, **TypeScript**, **Redux Toolkit**, and **Bulma**.  
Users can view products, filter by category, sort by price, and search by name. Clicking on a product opens a modal with detailed information.

---

## Features

- Fetch products from [FakeStore API](https://fakestoreapi.com/)
- Display product name, image, price
- Filter products by category
- Sort products by price (low → high, high → low, default)
- Search products by name
- Product details modal with larger image, description, and price
- Responsive layout for desktop and mobile
- State management with **Redux Toolkit**

---

## Technologies Used

- React.js
- TypeScript
- Redux Toolkit
- Bulma CSS framework
- Fetch API for getting products

---

## Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/hannachachacha/product-listing.git
   cd product-listing
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
The application will be available at: http://localhost:5173/product-listing/
